Why TCentral?

Create, consolidate and adjust the national trustee compliance work plan from this single centralized application.

TCentral is a web application that will allow OSB personnel in all regions to create Trustee Compliance Work Plans, track any changes made to each plan throughout the year, and store all plan related data in a central database where it can be accessed by reporting tools. With TCentral the excel spreadsheets currently being used for creating, submitting and consolidating compliance work plans will be eliminated.

Benefits

TCentral will reduce the time it takes to create and consolidate the national TC work plan.

Changes to the national TC work plan will be tracked, allowing for more accurate reporting. This will also help with resource allocation in future work plans.

Centralized data will allow all users to see the work plan as it really is, even as the plan changes throughout the fiscal period.
